# blockchain_optimization_frontend

## Project create
```
vue create blockchain_optimization_frontend
```

## Add artifacts
```
npm install element-plus --save
npm i vue-router@next
npm install axios
npm install echarts --save
npm install echarts@4.9.0
```

## Project setup
```
npm install
```
